How to run this project on your local machine:

1. Download and install XAMPP.

2. Clone or download this repository as a ZIP file.

3. Extract the folder, rename it to Tokusatsu, and move it to C:\xampp\htdocs\.

4. Open XAMPP Control Panel and start Apache and MySQL.

5. Go to http://localhost/phpmyadmin/. Create a new database named tokusatsu.

6. Import the SQL file located in the database/ folder into the newly created database.

7. Open your browser and go to: http://localhost/Tokusatsu/
